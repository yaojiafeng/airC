### 项目介绍
是的，这个就是当下很火热的空调微信小程序，简单而且能赚钱，给个数据大家慢慢体会
![image](https://github.com/yaojiafeng/airC/assets/32017295/4028615e-7db6-4d9f-b6d6-d379ee39bfc4)

### 分支
选择no-ad分支（无广告），换行自己的appid执行下面就可以跑起来拉
### 安装依赖

npm install -g @tarojs/cli

npm i  或者 yarn 

### 启动项目
npm run dev:weapp

### 打包项目
npm run build:weapp
