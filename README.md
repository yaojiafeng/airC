### 分支
选择no-ad分支（没有广告），换成自己的appid执行下面步骤就可以跑起来拉
### 安装依赖

npm install -g @tarojs/cli

npm i  或者 yarn 

### 启动项目
npm run dev:weapp

### 打包项目
npm run build:weapp
