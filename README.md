### 分支
选择no-ad分支（无广告），换行自己的appid执行下面就可以跑起来拉
### 安装依赖

npm install -g @tarojs/cli

npm i  或者 yarn 

### 启动项目
npm run dev:weapp

### 打包项目
npm run build:weapp
